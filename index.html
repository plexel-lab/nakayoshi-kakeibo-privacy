プライバシーポリシー

アプリ名：仲良し家計簿（以下「本アプリ」）／ 制定日：2026年6月1日

本プライバシーポリシーは、本アプリの運営者（以下「運営者」）が、本アプリの利用において取得する利用者の情報の取扱いについて定めるものです。利用者は、本アプリを利用することにより、本ポリシーに同意したものとみなされます。

1取得する情報

本アプリは、家計簿の記録・共有という機能を提供するために、以下の情報を取得・保存します。

区分内容保存場所
アカウント情報 メールアドレス、表示名、認証用の利用者識別子（UID）。Apple・Google・メールアドレスによるサインインの際に取得します。 Firebase Authentication / Cloud Firestore（Google）
家計簿データ 取引の金額・日付・カテゴリ・メモ・品目名・支払い方法、家計簿名、メンバー構成、招待コード、カレンダーの予定・通知設定、予算など、利用者が本アプリに入力した内容。 Cloud Firestore（Google）
レシート画像 レシート読み取り機能を利用する場合に、利用者が撮影・選択したレシート画像。利用者が自身の Google AI（Gemini）APIキーを設定した場合に限り、解析のため後述の外部サービスへ送信されます。 端末内（一時保存）／解析時のみ Google Gemini API へ送信
端末内データ Gemini APIキー、表示テーマ等の各種設定、未送信の保留レシート画像。これらは利用者の端末内にのみ保存され、運営者のサーバーには送信されません。 端末内（SharedPreferences／アプリ領域）

2利用目的

  • 家計簿の記録・集計・表示、および家計簿の家族・パートナー間での共有機能の提供
  • レシート画像からの品目・金額の自動読み取り（利用者が当該機能を有効にした場合）
  • カレンダー予定のローカル通知
  • 本アプリの保守・不具合対応、利用者からのお問い合わせ対応

3第三者サービスへの提供・連携

本アプリは、機能の提供のために以下の外部サービスを利用しています。各サービスにおける情報の取扱いは、それぞれの提供事業者のプライバシーポリシーに従います。

  • Google Firebase（Authentication / Cloud Firestore） — 認証および家計簿データの保存・同期に利用します。（Firebase のプライバシーとセキュリティ / Google プライバシーポリシー）
  • Sign in with Apple — Apple アカウントによるサインインに利用します。（Apple のプライバシーポリシー）
  • Google サインイン — Google アカウントによるサインインに利用します。（Google プライバシーポリシー）
  • Google Gemini API（generativelanguage.googleapis.com） — 利用者がレシート読み取り機能を有効にし、自身の APIキーを設定した場合に限り、レシート画像が解析のため送信されます。送信される画像の取扱いは Google の規約に従います。（Gemini API の利用規約）
  • Google AdMob — 無料プランの利用者に広告を表示するために利用します。AdMob は広告配信のために広告識別子等の情報を取得・利用する場合があります。iOS では、トラッキングの許可（App Tracking Transparency）を求め、許可しない場合は非パーソナライズ広告を配信します。有料（シルバー・ゴールド）プランでは広告を表示しません。（AdMob とプライバシー / Google 広告ポリシー）

上記のほか、運営者が利用者の同意を得ずに、取得した情報を第三者へ販売・提供することはありません。ただし、法令に基づく開示請求があった場合等、正当な理由がある場合はこの限りではありません。

4広告について

本アプリは、無料（ブロンズ）プランの利用者に対して Google AdMob による広告を表示します。広告配信のために、AdMob が広告識別子等を利用することがあります。iOS では端末の「トラッキングの許可」設定に従い、許可しない場合は利用者を追跡しない非パーソナライズ広告を配信します。有料プランにアップグレードすると広告は表示されません。

なお、本アプリは以下を行っていません。

  • 位置情報の取得
  • 利用者を識別する目的での独自のアクセス解析（アナリティクス）
  • 取得した家計簿データそのものの広告目的での利用・第三者への販売

5端末の権限

本アプリは、以下の目的で端末の機能へのアクセス許可を求めることがあります。許可は利用者が端末の設定からいつでも変更できます。

  • カメラ — レシートを撮影するため
  • 写真ライブラリ — レシートの写真を選択するため
  • 通知 — カレンダー予定の通知を表示するため

6データの保存期間と削除

家計簿データおよびアカウント情報は、利用者がアカウントを利用している間、保存されます。利用者は、本アプリの設定画面の「アカウントとデータを完全に削除」から、アカウントおよび関連するデータの削除を行うことができます。削除後、データの復元はできません。

なお、複数人で共有している家計簿については、他のメンバーが残っている場合、当該家計簿および他メンバーが入力したデータは引き続き保存されます。

7児童のプライバシー

本アプリは13歳未満の児童による単独での利用を想定していません。13歳未満の方が利用する場合は、保護者の同意と監督のもとでご利用ください。

8情報の安全管理

運営者は、取得した情報の漏えい・滅失・毀損の防止その他の安全管理のために、利用する外部サービスのセキュリティ機能（通信の暗号化、アクセス制御等）を適切に利用します。ただし、インターネットを通じた情報伝達の安全性を完全に保証するものではありません。

9本ポリシーの変更

運営者は、必要に応じて本ポリシーを変更することがあります。重要な変更を行う場合は、本アプリ内またはこのページにおいて告知します。変更後に本アプリを利用した場合、変更後のポリシーに同意したものとみなされます。
